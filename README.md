#Desafio Intelivix

Encurtador de URL

Requerimentos:
- Django 1.8.4
- Python 2.7+

Para rodar esse app, vá para o diretório onde foi feito o download e onde se encontra o arquivo manage.py:

EXECUTE
$ python manage.py syncdb
$ python manage.py runserver

Vá ao navegador e abra localhost:8000
